# KotlinChallengeApp
Repo for Android Kotlin Challenge app.

## Accomplishes the following: 
  - Reaches out to https://jsonplaceholder.typicode.com/photos API.
  - Model class using Moshi and Room, to parse the JSON and store the data locally.
  - RecyclerView with ViewBinding that displays the image and the image title.
    - OnClickListener for the imageView only that opens up the image on the device browser.
  
  
## Tech Stack (dependencies)
  Room, Coroutines, LifeCycle, Hilt + Dagger, Moshi, Retrofit, Coil-kt, MaterialDesign
